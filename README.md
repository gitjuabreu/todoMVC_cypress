# todoMVC_cypress
Automação de testes E2E no ToDoMVC utilizando Cypress, aplicando Page Object Model (POM) – projeto desenvolvido durante curso Udemy

Este repositório está realizando a automação Web do site todoMVC com o framework Cypress

# Tecnologias utilizadas:
- Node
- Cypress

# Testes criados:

- Adicionar texto
- Concluir Item
- Filtrar item
- Deletar Item
- Regressivo dos testes realizados
